# Objects Types / Data Types
- Number : 123, 3.132, 3+4j, ob11
- String : 'spam', "Bob's", """Docstring""", r"raw \n string"
- List : [1, 2, 3], ['abc', 34, 5.6], [], list(range(10))
- Dictionary : {'a': 1, 'b': 2}, dict(x=1, y=2)
- Tuple : (1, 2, 3), ('abc', 34, 5.6), tuple(range(10))
- Set : {1, 2, 3}, set(range(10))
- File : open('data.txt'), open('data.txt', 'rb')
- Boolean : True, False
- NoneType : None
- Bytes : b'abc', bytes(range(10))
- Bytearray : bytearray(b'abc'), bytearray(range(10))
- Frozen Set : frozenset([1, 2, 3])
- Array : array.array('i', [1, 2, 3])

- Functions , modules, classes, instances, etc., are also object types but are not typically referred to as data types.
